# India Map Guessing Game 

This project is a simple interactive web app where users try to guess Indian states using a map.

The user first enters their name, then types a state name and clicks on the map to match it. If the guess is correct, the score increases. At the end of the game, the result is saved and can be used for basic analysis.

## Features

- Interactive India map
- User can enter name and guess states
- Score increases for correct answers
- Works with lowercase and uppercase inputs
- Game data is stored in a CSV file
- Basic data analysis using Python

## Technologies Used

- Python (Flask)
- Pandas
- HTML, CSS, JavaScript
- SVG map

## How to run

1. Run the app:
   python app.py

2. (Optional) Run analysis:
   python analysis.py

## About the project

I built this project to practice web development and data analysis. It helped me understand how to take user input, validate it, store data, and analyze it using Python.
